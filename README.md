# Automation-testing
Selenium automation project for AI CERTs Learner Portal. Includes Java test scripts for key flows: sign-up, OTP, dashboard, and courses. Covers cross-browser validation using Eclipse IDE. Built for robust regression and easy extension.

Here’s enhanced README content for your **automation-testing** repository, with reference to your manual test case repository:

## Project Features

- Automated regression and functional tests using Selenium WebDriver and Java
- Clear locator strategies (XPath, CSS) for reliable script execution
- Covers major user journeys: sign-up validation, OTP flow, dashboard navigation, and course access
- Cross-browser compatibility: tested on Chrome, Edge, and Brave
- Easily maintainable code structure (modular classes, reusable methods, assertions)

## Getting Started

- Open in Eclipse IDE (import as Java project)
- Update WebDriver and environment setup according to your local configuration
- Execute scripts to validate new releases
